About the program
What is FIRST Robotics Competition?

FRC is sometimes called the "varsity sport for the mind." Teams design, build, and program a robot from scratch, then take it into a live competition arena against teams from across the country.

Skills you'll gain
What You'll Learn

Over the course of the program you'll pick up real, in-demand skills that go well beyond robotics.

CAD Design
Model and plan robot components using professional software before building anything physical.
Java Programming
Write real Java code to control your robot autonomously. One of the most in-demand languages out there.
Electrical Wiring
Wire motors, sensors, and control systems safely so your robot does exactly what you want it to.
Team Building
Solve real problems under real time pressure. Harder than it sounds, more fun than you'd think.
